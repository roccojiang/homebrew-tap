## How do I install these formulae?

`brew install roccojiang/tap/<formula>`

Or `brew tap roccojiang/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "roccojiang/tap"
brew "<formula>"
```
